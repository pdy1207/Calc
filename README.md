# A calculator

 - 나만의 계산기 앱을 만들어보자! 
 
    - 누적되서 보여주는 스트링 변수 변수 2개
    - 계산해야할 변수
    - 연산자 선택시 저장되어야하는 변수
    - boolean으로 인한 첫번째수냐 두번째수냐에 관한 true false 처리
    - 초기화 & 첫번째 수 입력 후 지워지고 두번째 수에 관한 처리
